# Latex-Installatie
In dit document wordt de installatieprocedure van LaTeX in Windows of OSX uitgewerkt. 
De lezer is volledig vrij om buiten deze programma's nog anderen te gebruiken.

Overzicht van de programma's:

## Platform-Agnostic
   * Overleaf
## Windows:
  * TeXStudio
  * TeXmaker
  * ATOM
## OSX
  * TeXnicle
  * TeXshop
